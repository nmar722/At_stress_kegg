# At_stress_kegg

## Transcript profiling by high throughput sequencing of Arabidopsis plants in response to a combination of salt and heat stresses.
Data was downloaded from [Expression Atlas web page](https://www.ebi.ac.uk/gxa/experiments/E-GEOD-72806/Downloads)

## KEGG Gene Set Enrichment Analysis
This worksheet follow a [Tutorial](https://learn.gencore.bio.nyu.edu/rna-seq-analysis/gene-set-enrichment-analysis/).

The Markdown html available at [web page](https://nmar722.github.io/At_stress_kegg/)
